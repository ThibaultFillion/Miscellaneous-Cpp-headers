I realized I've been re-implementing these simple geometry concepts and functions from scratch in each of my project,
so i decided to put those in a re-usable single header include to gain time and reliability.
Feel free to use it any way you want. The code is actually so trivial that i'm not sure any Licence should apply,
but the MIT-0 licence is the most permissive I know, so I put it in the file.

geometry.hpp:
contains classes for vectors in 2 and 3 dimensions (Vec2 and Vec3),
as well as a function to rotate a vector around an axis by some angle.

More function and files should be added later.
